## Theory
### NumPy:

NumPy is a fundamental Python library used for numerical computations. It provides support for:

- Multidimensional arrays (ndarray)
  
- Mathematical and statistical operations
  
- Fast and efficient computation compared to standard Python lists

### Key features:

- Array creation (array(), zeros(), ones())
  
- Indexing and slicing
  
- Mathematical functions (mean, sum, standard deviation)
  
### Pandas:

Pandas is used for handling structured data.

### Key structures:

- Series: One-dimensional labeled array
  
- DataFrame: Two-dimensional tabular data

### Operations include:

- Data selection (loc, iloc)
  
- Data cleaning and modification
  
- Statistical analysis (mean, max, min)
 
- Filtering and sorting data
  
### Applications:
  
- Data preprocessing
  
- Data analysis in real-world datasets
  
- Preparing data for machine learning
